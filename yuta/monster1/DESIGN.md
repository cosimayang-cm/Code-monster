# Feature Toggle 車輛系統

## 這是什麼？

一個車輛功能管理系統，讓你可以：
- 自由開關 12 種車輛功能
- 自訂配備組合
- 選擇預設車型

---

## 快速開始

### 方式一：自己組裝（Builder）

```swift
let myCar = CarBuilder()
    .withNavigation()       // 導航
    .withBluetooth()        // 藍牙
    .withSurroundView()     // 環景攝影
    .build()
```

### 方式二：選擇預設車型（Factory）

```swift
let car = CarFactory.create(.premium)  // 豪華款
```

| 車型 | 包含功能 |
|------|----------|
| `.basic` | 無選配 |
| `.standard` | 空調 + 娛樂 + 藍牙 |
| `.premium` | 標準款 + 導航 + 環景 |
| `.sport` | 豪華款 + 盲點偵測 + 前方雷達 |
| `.autonomous` | 全配（含自動駕駛） |

### 方式三：手動控制

```swift
let car = Car()
car.turnOnComputer()          // 開啟中控
car.enable(.airConditioner)   // 開冷氣
car.enable(.navigation)       // 開導航
car.disable(.navigation)      // 關導航
```

## 功能清單

| 功能 | 說明 | 需要先開啟 |
|------|------|-----------|
| 空調系統 | 冷氣控制 | - |
| 導航系統 | GPS 導航 | - |
| 娛樂系統 | 音樂、影片 | - |
| 藍牙系統 | 連接手機 | - |
| 倒車鏡頭 | 倒車影像 | - |
| 環景攝影 | 360 度環景 | 倒車鏡頭 |
| 盲點偵測 | 偵測兩側盲點 | - |
| 前方雷達 | 偵測前方障礙 | - |
| 停車輔助 | 自動停車 | 環景 + 盲點偵測 |
| 車道維持 | 維持車道行駛 | 導航 + 前雷達 + 引擎 |
| 緊急煞車 | 自動煞車 | 前雷達 + 引擎 |
| 自動駕駛 | 全自動駕駛 | 車道維持 + 緊急煞車 + 環景 |

## 設計說明

### 架構圖

```
┌─────────────────────────────────────────────────────┐
│                    使用方式                          │
├─────────────────────────────────────────────────────┤
│                                                     │
│   CarFactory          CarBuilder          Car       │
│   (預設車型)          (自訂組裝)        (手動控制)    │
│       │                   │               │         │
│       └───────────────────┼───────────────┘         │
│                           ▼                         │
│                   ┌──────────────┐                  │
│                   │     Car      │                  │
│                   │ ┌──────────┐ │                  │
│                   │ │ Feature  │ │ ← 功能開關       │
│                   │ │ Toggle   │ │   + 相依性檢查   │
│                   │ └──────────┘ │                  │
│                   └──────────────┘                  │
│                           │                         │
│              ┌────────────┼────────────┐            │
│              ▼            ▼            ▼            │
│         必要元件      選配元件      狀態管理         │
│         (4個)        (12個)                        │
└─────────────────────────────────────────────────────┘
```

### 使用的設計模式

| 模式 | 用途 | 位置 |
|------|------|------|
| **Feature Toggle** | 功能開關 + 相依性檢查 | `Car.swift` |
| **Builder** | 自訂組裝，Fluent API | `CarBuilder.swift` |
| **Factory** | 預設車型，批量生產 | `CarFactory.swift` |
| **Facade** | 統一介面 | `Car.swift` |

### 核心邏輯：相依性管理

**啟用功能時**，會檢查：
1. 中控電腦是否開啟
2. 引擎是否運行（部分功能需要）
3. 前置功能是否已啟用

**停用功能時**，會自動：
- 連鎖停用所有依賴此功能的其他功能

```
範例：停用「倒車鏡頭」
       │
       ├── 自動停用「環景攝影」（依賴倒車鏡頭）
       │         │
       │         ├── 自動停用「停車輔助」（依賴環景）
       │         │
       │         └── 自動停用「自動駕駛」（依賴環景）
       │
       └── 完成
```
