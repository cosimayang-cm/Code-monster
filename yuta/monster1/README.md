# Code Monster 作業1：Feature Toggle 車輛系統

這是 Code Monster 練習系列的第一個作業。

## 作業主題

實作一個車輛功能管理系統，包含：
- 12 種車輛功能的開關控制
- 功能相依性管理
- Builder 和 Factory 模式

## 檔案說明

- `DESIGN.md` - 系統設計文件
- `Package.swift` - Swift Package 配置
- `Sources/` - 原始碼
- `Tests/` - 測試檔案

## 執行測試

```bash
swift test
```

詳細設計說明請參考 [DESIGN.md](./DESIGN.md)
